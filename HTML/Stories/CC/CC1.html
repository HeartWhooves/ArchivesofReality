Chapter 0: Test Chapter

This chapter is not as part of the actual story, this is merely a test for how the story reader is going to work. I want to make sure it works and is readable so I can move on to developing further on the Website and then come back to fill it in as I actually start writing the stories. I still need to fill out the code to make the buttons work while also giving each story to come a section like this and also its own comment section.

I have absolutely no idea how im going to do the changing paragraphs. Would it be best to give each chapter its own page? Or maybe use Java script to change them out? I'm probably going to give each chapter its own page because then I can give each chapter its own comment section.

"But what are you going to do to manage all of those Pages?" You ask.

"Cry about it, probably," I reply. For it's with great sorrow that I know that I have no idea how I'm going to sort this all out unless I use folders. But folders come with the cost of needing me to fix every link between the pages. "I'm going to need to be less lazy," I sigh, resigned to my fate.

"Yes. Yes you will."

Soft sobs echoed through the halls of the archive.
